Read Before use

To link these files to work in your MPLAB environment, you need to add all of the .h files into the header folder
and you need to add all of the .c files into your source file folder in the environment. At this point the project
wont build. The crucial step as of right now is to:

-> Make sure you add in the provided linker, this is currently vital to see the project working.

==========================================
IF YOU JUST WANT TO USE THE LCD MODULE	==
==========================================

Just upload the lcd.h and lcd.c file into your project, as well as masterheader.h.
They have no dependancies and should just build without any further work on your part.

As per usual, check the lcd.h file for information on the modules available to you.
